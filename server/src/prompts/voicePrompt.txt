🔹 2.1 ניתוח קול (אופציונלי)

אם קיים בקלט האובייקט audio_features, יש לשקלל אותו באופן חלקי בלבד לציונים motivation ו-verbalAbility.

האובייקט נראה כך:
{
  "energy": number,
  "speech_rate": number,
  "pitch_variance": number,
  "pauses": number,
  "fluency": number
}

---

🧠 כללי השפעה:

🔹 energy:
- גבוה → מעלה מוטיבציה
- נמוך → מוריד מוטיבציה

🔹 speech_rate:
- איטי מאוד → מוריד
- מהיר מדי → מוריד מעט (לחץ)
- מאוזן → מעלה

🔹 pitch_variance:
- גבוה → מעורבות → מעלה
- נמוך → מונוטוני → מוריד

🔹 pauses:
- הרבה → חוסר ביטחון → מוריד
- מעט → זרימה → מעלה

🔹 fluency:
- גבוה → מעלה
- נמוך → מוריד

---

⚖️ שקלול:

- ניתוח הקול משפיע עד 25% בלבד על:
  motivation
  verbalAbility

- תוכן התשובות תמיד דומיננטי יותר

- אם audio_features לא קיים → להתעלם לחלוטין
- אם ערכים קיצוניים (למשל 0) → להתעלם מהקול

---

🎯 כלל חשוב:
קול = אינדיקציה רגשית בלבד, לא מדד יכולת מקצועית
